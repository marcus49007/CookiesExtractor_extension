# Privacy Policy


## Overview

This extension is designed to help users inspect and analyze browser cookies for educational, debugging, and testing purposes. We respect user privacy and are committed to transparent data handling practices.

## Information We Access

The extension may access:

- Cookies associated with websites the user explicitly chooses to inspect
- Basic website information required for cookie analysis
- User-generated exported files

The extension does **not** collect:
- Passwords
- Personal messages
- Payment information
- Browsing history outside user-initiated actions

## How Permissions Are Used

### Cookies Permission
Used to display and analyze cookies for websites selected by the user. Cookie access occurs only when the user actively uses the extension.

### ActiveTab Permission
Used to temporarily access the currently active tab after the user interacts with the extension.

### Downloads Permission
Used only to allow users to export and save cookie reports/files locally on their device.

### Host Permissions
Used to access website cookie data for sites the user chooses to inspect.

## Data Collection

This extension does not:
- Sell user data
- Share user data with third parties
- Use analytics or tracking systems
- Collect data in the background
- Transmit cookie data to external servers

All processing is performed locally within the user's browser unless the user explicitly exports data.

## User Control

Users have full control over the extension and can:
- Choose which websites to inspect
- Delete exported files at any time
- Disable or uninstall the extension whenever desired

## Data Storage

The extension does not permanently store sensitive user data on external servers. Any exported data is stored only on the user's local device.

## Third-Party Services

This extension does not use third-party advertising, analytics, or remote tracking services.

## Security

We take reasonable measures to ensure the extension operates securely and only within the permissions granted by the user.

## Changes to This Policy

This Privacy Policy may be updated from time to time. Continued use of the extension after updates constitutes acceptance of the revised policy.

## Contact

If you have questions regarding this Privacy Policy, please contact the developer through the extension listing page.
